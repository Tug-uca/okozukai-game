# Phase 1: ゲーム企画 - 作業計画

## プロジェクト概要
小学1年生向けの「おこづかいシミュレーションゲーム」の企画書を作成します。
このゲームは、子どもたちが「作戦を立て→実行する→結果から学ぶ」という体験を通じて、実現可能な計画の立て方を学ぶことを目的とします。

## ホスティング制約（GitHub Pages）
- サーバーサイド処理不可
- データベース不可
- ユーザー認証不可
